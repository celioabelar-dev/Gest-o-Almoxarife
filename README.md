## ✨ O que este sistema faz?

* **Gestão de Ferramentas:** Cadastro e acompanhamento do estado de conservação de cada item.
* **Movimentações:** Registro seguro de entradas, saídas e devoluções.
* **Relatórios e Histórico:** Rastreabilidade completa para saber quem pegou qual ferramenta e quando.

## 🛠️ Tecnologias Utilizadas

* **Google Apps Script:** Lógica de back-end.
* **HTML / CSS:** Construção da interface de usuário.
* **Google Sheets:** Banco de dados integrado.
